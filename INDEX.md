## 🆘 Getting Unstuck

**Step 1:** Search [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for your issue

**Step 2:** If not found, check relevant guide:
- Setup issue → [SETUP.md](SETUP.md)
- Code issue → [CONTRIBUTING.md](CONTRIBUTING.md)
- Config issue → [CONFIG.md](CONFIG.md)
- Command issue → [QUICK_REFERENCE.md](QUICK_REFERENCE.md)

**Step 3:** Review project [README.md](README.md) and [API.md](API.md)

**Step 4:** Check GitHub Issues (might be reported before)

**Step 5:** Ask in team chat with:
- What you tried
- Error message
- Relevant logs
